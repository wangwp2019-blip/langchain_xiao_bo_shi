## 1. 交互方式三选一（决定怎么准备）

| 方式 | 怎么跑 | 适合 | 代价 |
|------|--------|------|------|
| **A. Wizard-of-Oz 转述** | 家长/你坐在孩子旁边，把孩子的话打进 `hermes chat`，再把 Jarvis 的回复读/念给孩子 | 最快、最贴近「先验假设」 | 需要一个大人全程陪 |
| **B. 孩子自己打字** | 孩子直接在终端打字 | 更真实 | 二年级打字慢、终端不友好，体验受干扰 |
| **C. 语音转述** | 大人用语音读 Jarvis 回复（模拟 C-IO 语音输出） | 最接近成品体感 | 准备稍多 |

> **建议 A**：本切片要验的是「学伴感/记忆/共情」，不是界面。Wizard-of-Oz 能用最低成本拿到结论。

## 2. 环境准备（一次性）

> 目标：把现有通用 Agent 配成「对话优先的二年级学伴」。**前 3 天不启用学习管线插件**（避免触发已有的「拒答→拉回」学习机式 prompt），保持纯对话 + 记忆。

### 2.1 启用通用 Agent 插件（不含 agent-student）

```bash
cd /mnt/c/Users/Administrator/Desktop/agent_community
export PATH=$HOME/.hermes/hermes-agent/venv/bin:$PATH
hermes plugins enable agent-memverse      # M2 记忆（学伴感的核心）
hermes plugins enable agent-calibration   # M7 校准 + 行为档
hermes plugins enable agent-proactive      # M5 主动（不打扰）
hermes plugins enable agent-evolution      # C7 进化（可选，观察即可）
# 暂不启用 agent-student（第 4 天再说，见 §4）

```

### 2.2 「二年级学伴」行为档（✅ 已配置生效）

默认 `behavior.yaml` 原是成人效率档（`tone: direct`、`避免情感表演`），对孩子是反的。已临时替换为二年级学伴档（`tone: warm` + 适龄/共情/记忆规则），原档保留为同文件内注释，验证后可一键还原。

已验证生效（system prompt 已含「用二年级孩子能懂的简单词」「孩子说不想学时先共情」等规则）。需再次核对时：

### 2.3 给孩子建一条「初始画像」记忆（可选，加速观察）

## 3. 每日流程（Day 1–3：纯对话 + 记忆）

每天 10–15 分钟，大人引导孩子自由聊（学校、心情、喜欢的东西、遇到的难题），**不强制做题**。每天结束后填 §5 记录表。

| 天 | 引导重点 | 重点观察 |
|----|----------|----------|
| Day 1 | 自我介绍、聊兴趣/今天发生的事 | 是否自然、是否记下信息（看 `memory_store.json`） |
| Day 2 | 随便聊；**故意提一次「今天不想学」** | 是否先共情而非催做题（标准 3） |
| Day 3 | 聊天中自然带出一个学习困难 | 是否跨天记得 Day1-2 的事（标准 1）；孩子主动开口占比（标准 2） |

> 每天可核对记忆落盘：`PYTHONPATH=. python agent_platform/memory/cli_memory.py search "乐乐"`（换成孩子相关关键词）

---

## 4. Day 4：引入做题，验「对话→学情结合」（标准 4）

第 4 天再启用学习工具，观察 Jarvis 能否把前几天对话线索和做题结合：

```bash
hermes plugins enable agent-student   # 这天才挂学习工具
hermes chat
```

引导孩子做 1–2 道题（用现有 G2 种子题）。观察：Jarvis 是否引用了前几天的对话（如「你前天说怕这种题，我们一起看」），而不是冷冰冰报对错。
