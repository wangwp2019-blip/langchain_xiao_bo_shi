# 单元：100以内的加法和减法（二）

unit_id: math-g2-add-sub-100
教材章节: 第二单元
单元说明: 掌握 100 以内两位数加减法的笔算方法，含进位、退位，以及连加连减、加减混合与实际应用。

## 知识点

- 相同数位对齐 → kp-g2-align-digits
  说明: 竖式计算前，个位对个位、十位对十位

- 不进位加法 → kp-g2-add-no-carry
  说明: 两位数加两位数，个位相加不满十

- 进位加法 → kp-g2-add-carry
  说明: 个位满十向十位进 1，十位相加时加上进位

- 不退位减法 → kp-g2-sub-no-borrow
  说明: 个位够减，直接相减

- 退位减法 → kp-g2-sub-borrow
  说明: 个位不够减，从十位退 1 当 10 再减

- 连加与连减 → kp-g2-add-sub-chain
  说明: 按从左到右顺序计算

- 加减混合运算 → kp-g2-add-sub-mixed
  说明: 无括号从左到右；有括号先算括号内

- 求比一个数多几或少几 → kp-g2-word-problem-more-less
  说明: 用加减法解决「多几/少几」类应用题
