# 单元：句子与标点基础

unit_id: chinese-g2-sentence-basic
教材章节: 全册句子专项 / 语文园地（归纳）
单元说明: 能判断句子是否完整，正确使用句号、逗号、问号、感叹号，初步连词成句。

## 知识点

- 陈述句与句号 → kp-g2-punct-period
  说明: 陈述一件事的句子末尾用句号

- 疑问句与问号 → kp-g2-punct-question

  说明: 表示疑问的句子末尾用问号

- 感叹句与感叹号 → kp-g2-punct-exclaim
  说明: 表达强烈感情的句子末尾用感叹号

- 逗号与句内停顿 → kp-g2-punct-comma
  说明: 句子内部的一般停顿用逗号

- 连词成句 → kp-g2-word-order
  说明: 把词语排列成通顺、完整的句子

- 句子是否完整 → kp-g2-sentence-complete
  说明: 能判断句子是否缺少主要成分，并补全

# 单元：词语积累与搭配

unit_id: chinese-g2-words-collocation
教材章节: 各单元词语积累（归纳）
单元说明: 积累常用词语，掌握数量词、形容词与名词的搭配，能说简单通顺的句子。

## 知识点

- 量词与名词搭配 → kp-g2-measure-word
  说明: 如「一只」「一条」「一座」等常见量词

- 形容词与名词搭配 → kp-g2-adj-noun
  说明: 如「碧绿的衣裳」「茂密的森林」

- 近义词与反义词 → kp-g2-synonym-antonym
  说明: 区分并运用常见近义、反义词语

- 多音字辨析 → kp-g2-polyphone
  说明: 同一汉字在不同词语中的读音

- 用给定词语说一句话 → kp-g2-word-to-sentence
  说明: 运用本单元词语组织通顺句子
