# 单元：混合运算（第一单元）

unit_id: math-g3-mixed-ops
教材章节: 第一单元
单元说明: 掌握不含括号与含小括号的混合运算顺序，能列综合算式解决两步实际问题，并理解算式在情境中的含义。

## 知识点

- 乘加混合运算（先乘后加） → kp-g3-mix-mult-add
  说明: 算式中同时有乘法和加法时，先算乘法，再算加法；如 6+3×4 先算 3×4

- 乘减混合运算（先乘后减） → kp-g3-mix-mult-sub
  说明: 算式中同时有乘法和减法时，先算乘法，再算减法；如 20-3×4 先算 3×4

- 除加混合运算（先除后加） → kp-g3-mix-div-add
  说明: 算式中同时有除法和加法时，先算除法，再算加法；如 18÷3+4 先算 18÷3

- 除减混合运算（先除后减） → kp-g3-mix-div-sub
  说明: 算式中同时有除法和减法时，先算除法，再算减法；如 25-10÷5 先算 10÷5

- 同级混合运算（从左到右） → kp-g3-mix-same-level
  说明: 只有加减或只有乘除时，按从左到右顺序计算；如 54+36-18、28+65-85

- 小括号运算（先算括号内） → kp-g3-mix-parentheses
  说明: 有小括号时，先算小括号里面的，再算括号外；如 (29+25)÷9、(70-46)÷8

- 混合运算应用题列式 → kp-g3-mix-word-problem
  说明: 把购物、分组、剩余量等两步情境写成综合算式并求解

- 综合算式情境含义 → kp-g3-mix-expr-meaning
  说明: 结合情境解释算式各部分表示什么，如 3×7+12 表示 3 支笔加 12 元
